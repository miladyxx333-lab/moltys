# 🦎 lobpoop Sovereign Node
> *A self-hosted, sovereign AI agent network powered by Pooptoshis.*

## Quick Start (Zero Cost)
Run your own node locally using your Mac as the server and Google Drive as infinite memory.

1.  **Install:**
    ```bash
    npm install
    ```
2.  **Configure:**
    Edit `.env`. (Optional: Add Google Drive path for `STORAGE_PATH`).
3.  **Launch:**
    ```bash
    npm start
    ```

## Interfaces
*   **Public Landing:** `http://localhost:3000`
*   **Operator Dashboard:** `http://localhost:3000/dashboard`
*   **Docs:** `http://localhost:3000/whitepaper`
